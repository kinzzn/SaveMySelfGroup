# MAZZEL
### 个性 × 团体 = 最强的8人出道

# KAIRYU

### 想到达那种“让人觉得原来还有这种歌手的存在”的位置

参加过《THE FIRST》，凭借独一无二的歌声引发话题的KAIRYU。带着非同一般的热情成为BMSG的练习生，在《MISSIONx2》中，他那种看得很远的视角也得到了SKY-HI的称赞。不过，他的目标其实并不是“出道”，而是在更远的地方。

我开始认真想要当艺人的那一天，是2015年7月25日。这个日期我记得非常清楚。当时是初三，我和姐姐一起去京瓷巨蛋看「SMTOWN」（韩国SM娱乐公司的活动）。真的被震撼到了。回到家以后，我马上就跟父母说：「我以后要做那个。」其实我一直都很喜欢唱歌，但从来没有在朋友面前唱过一次。怎么说呢……就是不太想（笑）。后来我去了韩国，当了半年练习生，然后回到了日本。在那之前，我看的基本都是韩国的音乐圈。后来看到「THE FIRST」的招募页面，那些话真的一句一句地往我心里扎。我当时就觉得：“就是这里了。”

在《MISSIONx2》的时候，我、SEITO，还有RAN是先开始训练的人。后来新的候补生陆续加入，其实有很多地方需要顾虑——比如他们会怎么看我们之类的。SKY-HI说过「KAIRYU的轴一直没有动摇」，但其实我自己并没有特别意识到这一点。要说的话，其实是因为出道本身并不是我的目标。我真正的梦想，是在这个行业里取得成功。所以也许正因为这样，我才没有动摇。经历了这个选秀之后，我其实也没有觉得自己发生了特别巨大的变化。不过，因为是在BMSG，所以我可以很自然地把最真实的自己拿出来。

我觉得自己成长最多的一次，是在和其他候补生合流之前，我们三个人最后一次合宿的时候。那一周基本就是每天疯狂练习，Money老师几乎一直陪着我们教。那段时间我真的感觉自己的舞蹈进步特别大。那次像是把基础彻底打牢了。等到后来大家一起合宿的时候，就能在这个基础上继续开花。还有一个人让我受到很大刺激，就是EIKI。我从小就是那种有点爱逞强的人，很在意别人怎么看自己。但EIKI身上有一种……很“泥土味”的感觉吧。不是刻意装出来的帅，而是一种很真实、不修饰的帅。对我来说真的很耀眼。

### 出道之后，反而出现了更具体的课题

真正宣布出道的那一刻，其实我整个人是有点飘着的。因为当时刚一下子听到了很多信息，团名、人数之类的，而且我还是第一个被叫到名字的人。说实话，当时脑子有点空白。从决定要出道开始算，这 7年的时间里，其实有很多很辛苦的阶段。甚至有些苦到没办法公开说。因为那是一个太大的目标了，所以在听到出道的时候，我反而没有马上那种「太好了！！」的感觉。真正有实感，其实是最近才开始的。慢慢开始看到排行榜、播放次数这些数字，也能听到支持我们的人的声音。最近才越来越强烈地感觉到：“啊，一切真的要开始了。”出道决定之后，我反而在表演方面开始有更多具体的想法。比如“这里想这样做。”“那里可以那样处理。”这些想法会不断冒出来。所以我一边想象这些画面，一边拼命继续练习。根本没有时间飘飘然。而且出道这件事，是会用数字直接给出答案的。所以它既让人开心，也真的很可怕。作为团体，我们的梦想是——站上京瓷巨蛋。因为那是让我第一次产生梦想的地方。我想成为那种能站在那里表演的艺人。

如果说个人目标的话，我还是想把唱歌做到更极致。我很喜欢HIP-HOP和R&B，也希望能和这个圈子的艺术家有更多交流。同时也希望，有一天当各种喜欢音乐的人听到的时候，会说“原来还有这样一个歌手。”我想到达那样的位置。

MZ-02

# NAOYA

### 得到认可，摆脱负循环，终于获得作为艺人的自信

NAOYA拥有扎实的舞蹈与歌唱实力、俏皮的比心动作，以及极具吸引力的魅力。不过令人意外的是，他坦言自己“完全没有自信”。而这种不自信，正是因为过去经历了太多挫折。

在参加《MISSIONx2》之前，大概有8年的时间，我一直在追逐“成为世界第一艺人”的梦想。但一直都没有什么成绩。周围的大人也很少称赞我的表演，所以我渐渐觉得，自己既没有舞蹈实力，也没有歌唱实力，表现力也不够，甚至连魅力都没有。慢慢地就变成一种状态：做什么都不顺利。练习再多，一到正式表演就紧张得发挥不出来，完全陷进了一种负循环。但在《MISSIONx2》的评审中，我非常尊敬的SKY-HI每一次都会称赞我。那让我慢慢意识到——只要继续努力，舞蹈、歌唱，还有表现力，都是可以不断成长的。从那之后，我就更想继续变得更强。

尤其是在训练营的初期选秀阶段，我自己也很明显地感受到成长。表演结束后，我在镜头前表现得好像一点也不后悔的样子，但其实内心是非常不甘心的。因为我知道自己明明可以练得更好。结果在公布结果时，我第一个被叫到名字。SKY-HI对我说「表演真的很好。」「表情也非常有表现力。」听到这些话的时候，我很开心，但同时也有点不满足。因为我自己还没有完全搞清楚自己的课题是什么。所以后来我把其他成员被指出的问题也拿来对照自己，试着找出“如果是我，会有什么问题”。之后在评审时，当我被称赞这种发现问题的方法时，我就确信了自己现在走的这条路，是对的。那一刻，我终于获得了过去8年都没有拥有过的——作为艺人的自信。也正因为这样，我才能带着信心面对最后的表演。

### 想成为每一首歌都能变换颜色的“变色龙”

在《MISSIONx2》中遇到的人，也给了我很多启发。有拼命专注于舞蹈和歌唱的成员，也有专业的老师们。正因为我一直觉得自己还有很多不足，所以总是想着“只要是能学的地方，就全部吸收。”在成员当中，我特别从RAN身上学到了很多。不仅是舞蹈、歌唱、表演这些技术层面，还有他那种认真努力的态度，以及对成员们很亲切、很温柔的性格。我觉得这些都是作为一个艺人非常重要的部分。

现在终于可以正式朝着梦想出发了，但成长当然不能就此停止。除了舞蹈和歌唱之外，我还想继续提升自己的表现力。就像预出道曲《MISSION》和出道曲《Vivid》一样，我希望自己能成为那种每一首歌的氛围都可以完全改变的“变色龙型艺人”。所以接下来我也想接触更多不同类型的音乐，研究各种组合的表现方式。不管是男性还是女性艺人，不管是什么风格，我都想去学习，然后不断展现新的自己。因为我们的成员每个人个性都完全不同，所以也能尝试很多不同类型的音乐。而且不管是什么领域，我都希望我们能拿出最好的表现。组合成立之后，我们成员之间也经常聊天。慢慢才发现，其实每个人都经历过很多挫折。虽然大家的个性、技能、喜欢的音乐都不一样，但我们有一个共同的目标。我觉得这一点真的很有意思。现在越来越能感觉到8个人正在慢慢变成一个整体。而这样的感觉，也让我变得比以前更加有自信了。

# RAN

### 经历了无可替代的两年岁月 终于站上了真正的起跑线

在清楚知道“哪个地下城里会藏着什么敌人”的情况下再次挑战——RAN用RPG游戏来形容自己参加《MISSIONx2》的心境。这像是一场“二周目开局”。而那些敌人，其实正是他自己的弱点。BE:FIRST成员们的努力、《BMSG FES'22》的舞台，以及SKY-HI的话语——这一切，都成为推动他前进的力量。

在「THE FIRST」最终审查落选的时候，说实话，我完全没想过自己会落选，也不愿意去想这种可能。当时我在想：接下来到底该怎么办。就在那个时候，给我机会的人还是SKY-HI。我以练习生的身份和BMSG签约，过了一段时间之后，《MISSIONx2》也开始启动了。不过那时候，我其实完全没有余力去考虑未来。再加上途中REIKO离开了。对当时的我来说，他是唯一真正了解我内心状态的人，所以那时候我又重新感到不安。但后来KAIRYU和SEITO加入了。尤其是SEITO，我们本来就因为跳舞认识，他真的给了我很大的支持。在和大家一起努力的过程中，我也越来越强烈地觉得自己想和这些人一起继续走下去。就在这样的心情中，新的选秀正式开始了。

8月的训练营形式和「THE FIRST」几乎一样，所以说实话，非常紧张。从15人开始的训练营，对我来说就像游戏里常说的“二周目开局”。因为我已经知道接下来会发生什么。但同时，我也必须继续成长。而新聚集起来的成员水平也非常高，我也有可能会被吞没。每一次审查我都非常紧张，等待念到自己名字的时候，真的感觉像是没法呼吸一样。心脏都快要炸开了。老实说，那种感觉我真的不想再经历一次了（笑）。


### 因为成员的存在 让我觉得“不能在这种地方停下”

能够克服上一次落选时自己的弱点，并顺利走完《MISSIONx2》，我觉得有几个原因。其中一个，是看到BE:FIRST成员们那么努力，还有选秀参加者们积极向前的姿态，我自己的情绪也被一点一点带动起来。另一个，是在选秀过程中经历了BMSG FES。那次经历，让我作为艺人的自我意识又往前迈进了一步。还有一点，是我想SKY-HI曾经对我说过的话。他说“在想象未来的时候，试着把画面想得具体一点。”从那个时候开始，我感觉像是一直笼罩的雾终于散开了一样，前方的道路慢慢变得清晰。然后我意识到，如果想走到那个未来，现在必须做的事情，就是在这场选秀里继续走下去。

MAZZEL的8个成员，每个人都有完全不同的背景。在和大家聊天、问他们为什么想成为艺人的过程中，我慢慢发现，另外7个人，其实也和我一样，经历了很多事情，才来到这里。当大家带着同样的志向，准备一起走接下来的路时，我突然觉得，如果我现在还在输给自己，那就太不像话了。走到今天这一步之前，我也有过很多次不甘心的时候。甚至也有几次想过：要不要就放弃算了。虽然真的只有几次（笑）。但最后我还是会告诉自己：我现在还远远不够。如果在这种地方就停下，那以后也不可能再继续往前走。所以还是要继续努力。

在最终审查被选中的时候，如果用一句话来形容，那就是“终于啊。”（笑）这两年的时间，对我来说，是人生中非常浓密的一段经历。我想，大概在今后的人生里，也很难再有比这更重要的两年了。终于到了可以回报那些一直支持我的人的时候。而且最让我高兴的是，终于能让一直支持我的SKY-HI看到我真正活跃的样子。作为个人来说，我希望MAZZEL能成为那种，无论几年、几十年之后回头看，仍然会被人记住、被人觉得“很酷”的艺人。

# SEITO

### 在「Mx2」中认识自己、战胜自己 想成为MAZZEL的“爆发力”

SEITO用充满力量的舞蹈表演吸引观众，同时也因为温和的说话方式和幽默感而深受喜爱。不过，在加入BMSG之前，以及被选为MAZZEL成员之前，他的笑容背后其实经历过很多挣扎。

看到「THE FIRST」里聚集了那么多天赋惊人的人时，我真的非常着急。其实在那之前，我就已经经常找SKY-HI商量很多事情，也听他说过要创立BMSG。我当时一直对他说「我一定会来到这里的。」就这样一次又一次地主动去争取，最后终于以Trainee的身份加入了。不过之后一直没有什么能公开露面的机会，那段时间真的很煎熬。所以当听说要组建第二个组合的时候，我的第一反应是——终于来了。怎么说呢……也有一种松了一口气的感觉。虽然心里也明白，这一切才刚刚开始，但能终于站上起跑线，本身就让我非常开心。

不过说实话，最开始的时候我一直都很焦虑。因为过去发生过很多事情，那段时间我的精神状态其实也有点被压垮。像RAN和REIKO（レイちゃん），他们已经经历过很多舞台了。而KAIRYU的歌又那么厉害。其实我一直觉得“我有舞蹈。”这一点我本来是非常有自信的。但是不知道为什么，就是一直在焦虑。我本来是那种很乐观的性格，几乎不会因为事情消沉。可那时候我第一次意识到：原来自己也可以这么脆弱。特别是在《FANTASY》那段时期，我的心理状态真的糟透了。我已经把人生都押在这件事上了。而且从离开之前的事务所、来到BMSG到那个时候，也已经过了差不多两年。所以脑子里一直在想一些很极端的事情“如果这次失败，一切就结束了。”结果就一直往不好的方向去想。

### 看了「BMSG FES」上RAN的舞台后 心态彻底改变

让我一下子走出这种状态的契机，是在下一首课题曲《Get Down》之前，我和KAIRYU一起去看了BMSG FES'22。那天最大的冲击，是看到RAN站在舞台上的样子。真的太震撼了。我当时既羡慕，又不甘心。我也很清楚，以现在这样的状态，我是绝对站不上那个舞台的。但正因为这样，我反而燃起来了。我当时想：我也一定要站上那个舞台。既然这样，那就只能拼命去做。那一刻，我的情绪朝着一个非常好的方向燃烧了起来。有时候我也会想，如果没有那一天，我现在会变成什么样呢。

在最终审查时，当我被叫到名字、成为成员的时候，我只记得自己松了一口气。不过SKY-HI后来对我说：「其实也考虑过一个没有SEITO的组合。」听到这句话的时候，我心里想的是——果然被看出来了啊。我的弱点，其实早就被SKY-HI看穿了。但同时，他也看到了我之后一点一点改变的过程。想到这一点的时候，真的有很多情绪一下子涌上来。出道确定之后，我马上给家里打了电话。我妈妈当时哭得特别厉害。毕竟一直以来，最担心我、也被我添过最多麻烦的人，就是她。

通过《MISSION×2》，我觉得自己真的认识了自己，也战胜了自己。同时，我也下定了决心——要以MAZZEL成员的身份，真正站到舞台前面。接下来能做的，就只有努力做出好的音乐，把它传递给大家。我的想法也慢慢变得很坚定。而且我还给自己设了一个小小的“MISSION”（笑）。在拍艺术照之前的3周时间里，我减掉了5公斤。接下来我也想把这个状态保持下去，继续打磨自己。成员们经常说，我在表演时释放出来的力量，比别人都要更大。我觉得这也是我的一个优势。而且在舞蹈方面，我还有很多还没展示出来的东西。
所以，我想成为MAZZEL的“爆发力”。

# RYUKI

### 想在MAZZEL做出让人震惊的音乐 个人来说也对演戏很感兴趣

RYUKI总是带着笑容。既有18岁的可爱感，但一旦站上舞台，就会展现出节奏凌厉的RAP和充满力量的舞蹈，这种反差正是他的魅力。SKY-HI也对他寄予厚望，称他是“像秘密武器一样、能够成为组合核心的存在”。

我是在看了LDH的演唱会之后，开始想成为艺人的。小学三年级的时候，我就进了舞蹈学校。后来参加了选秀「iCONZ」，不过结果不太理想。之后我也去参加过一些演技方面的试镜。BE:FIRST的MANATO跟我是同一所舞蹈学校出来的。我当时落选的时候，还打电话找他商量过。其实我本来就有在看「THE FIRST」，不过有一次刚好在《スッキリ》节目里看到BE:FIRST的现场演出影像，真的帅到不行。从那时候开始，我就越来越想“想在BMSG做出最棒的音乐，然后做最棒的现场演出。”所以我就报名了。

在《MISSIONx2》开始之前，我其实挺不安的。毕竟这是一个水平很高的事务所，也不知道自己会被怎么评价。但同时我也一直告诉自己：“绝对要通过。”后来我是在看节目播出的时候才知道，原来SKY-HI形容我是“飞行道具”（秘密武器）。听到的时候真的特别开心。因为我一直希望自己能成为那种，一登场就能把整首歌的气氛完全改变的人。就像一个冲在最前面的先锋一样。所以当我发现自己的想法和SKY-HI的评价完全一致的时候，我心里真的在想“太好了！”其实我学RAP大概只有一年左右。但我很喜欢那种很猛、很冲的表演风格，所以包括RAP创作在内，我以后也想继续把它做到更厉害。

在选秀过程中，我最明显感觉到自己成长的一次，是在《FANTASY》的时候。那是一首唱成熟恋爱的歌，对我来说真的很难。而且我之前也没有唱歌的经验，所以跟声乐老师学了很多。最后拿到了第3名。不过说实话，我当时其实是想拿第1名的，所以还是很不甘心。在选秀后期，还有一件让我印象特别深的事情。我和EIKI一起哭了。因为我在成员里算年纪比较小的，有时候练习不顺利的时候，我们自己也没办法很好地把大家带起来。工作人员也对我说过：“年龄其实没有关系。”听到那句话的时候，我真的觉得自己有点丢脸。但那个时候EIKI的存在对我来说真的很重要。我们互相鼓励，也有一种一起变强的感觉。


### 学会用更积极的方式接受别人的意见

通过这次选秀，我觉得自己最大的变化是——开始能够用更积极的方式去接受别人的意见。以前如果有人提出不同的想法，我可能会直接觉得：“但我觉得是这样啊。”

# EIKI

### 「Mx2」时期是拼命抓住机会的心情 想用歌声成为“MAZZEL的脊梁”

### 被NAOYA影响的“表演方式”

选秀刚开始的时候，我其实很在意周围的人。但后来我慢慢明白，大家不是敌人。虽然是竞争对手，但也是伙伴。明白这一点之后，我就能对大家敞开心扉，也能依赖大家，甚至把自己的脆弱也展现出来。没有舞蹈经验这件事，真的让我非常不甘心。但那也不能成为借口。不过我心里也会想：“但我是初学者，这也是事实。”这算是我唯一一点点的小小安慰，也正因为这样，我才能把自己的自尊放下来。最重要的是，我真的很想进入BMSG。所以不懂的就只能去问，只能去做。当时的心情就是——拼命抓住这个机会。

对我来说一个很大的课题，就是“什么叫做表演”。在《FANTASY》那一关和我同队的NAOYA，让我真正理解了这件事。看着他的时候我才明白——原来表演不是“只是唱歌、只是跳舞”。“原来表演是这样的啊！”如果不能传达给看的人，那就没有意义。NAOYA身上有一种很难用语言说清楚的东西，我真的被他深深影响了。他说自己会研究什么角度看起来最帅，也会很注意视线的使用方式。但在我看来，更厉害的是他整个人进入音乐里的那种能力，简直像怪物一样。我也开始模仿他，慢慢去注意这些东西。不过现在再回头看当时的自己……那时候我还觉得自己表现得挺不错的，但现在看就会想——原来当时我只能做到这样而已啊。

出道确定的那一瞬间，我的心脏已经不是“砰砰跳”那种程度了，反而是整个人变成了“空白”。因为在参加最终审查的10个人里面，我是第8个被叫到名字的，也是最后一个。所以那一刻，我脑子里一直想着那两个没有被叫到名字的人。说实话，比起开心，我反而没办法很自然地笑出来。之后一边练习，一边搬到东京生活。真正让我有“啊，我真的出道了”的感觉，是在预出道记者会的时候。那一刻我觉得，看着我们的大家和我们之间的时间轴，好像终于对上了。接下来我也会抱着“我要把舞蹈变得更强”的决心继续努力。唱歌我也认为是我的武器，所以我想用歌声成为“MAZZEL的脊梁”。希望有一天大家会说：“MAZZEL少不了EIKI的歌声。”我想成为那样有存在感的人。而且最重要的是，能和这8个人一直一起活动下去——这就是我的梦想。
